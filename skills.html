Software-Projekt 2023:

Im 4 und 5 Semester müssen die Studenten an einem ganzjähriges Softwareprojekt teilnehmen und so erste praxisnahe Erfahrungen sammeln. In einem Team werden alle Schritte einer Softwareentwicklung selbstständig geplant und durchgeführt, wobei das Wissen der vorherigen Semester kombiniert und praktisch angewendet wird.

Ich habe mich dazu entschlossen an dem Projekt Entwicklung von Informationssystemen mit SCRUM teilzunehmen. Unser Team hat dabei die Aufgabe erhalten eine Webseite für das sammeln von Spenden zu erstellen. Gefordert war das sich Spender und Spendenorganisationen einen Account anlegen können und mittels z.B. Paypal Spenden an die Organisationen entrichtet werden können. Spender und Organisationen sollen diese Spenden in Form von Quittungen nachvollziehen können.


Technische Umsetzung:
  • Bitbucket: Versionsverwaltung von Dateien
  • Software Ideas Modeler: Erstellen von Architektur Diagrammen
  • Jira und Confluence: Planung und Dokumentation des Projekts
  • Bamboo und JUnit: Zum Testen des Programmcodes
  • GlassFish: Anwendungsserver
  • Postgres: SQL Datenbank Anwendung
  • Java Server Faces: Java-Framework
  • PrimeFaces: UI-Framework

Die Webanwendung wurde mit dem Framework Java Server Faces erstellt. Bei Java Server Faces handelt es sich um ein serverseitiges Framework. Java Server Faces verbirgt die Komplexität der Webanwendungsentwicklung. Für die Speicherung von Daten wurde die Java Persistence API verwendet. Die damit in der Datenbank persistierten Daten können dann über die Frontend Schnittstelle zur Verfügung gestellt werden. Die Benutzeroberfläche wurde mittels des UI-Frameworks PrimeFaces erstellt.
