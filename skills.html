Berufserfahrung

Ausbildung zum Fachinformatiker:

Da ich auf der Statseite unter dem Bereich Fähigkeiten bereits darauf eingegangen bin, dass ich während meiner Tätigkeit bei der Go-S3 Business Solution im Bereich Support, Consulting und Qualitätsmanagement gearbeitet habe. Möchte ich an dieser Stelle genauer auf die im Unternehmen verwendeten Technologien und den Arbeitsablauf eingehen.


Arbeitsbereich:

Die Go-S3 Business Solution vertreibt eine eigens entwickelte Software namens ECCOSoft CMMS. Diese Software wird in den Bereichen Instandhaltung, Abfallmanagement und Dokumentenmanagement eingesetzt. Es handelt sich um eine Modular aufgebaute Software welche für die von Unternehmen jeweils benötigten Bereiche individuelle eingerichtet werden kann. Das System wurde mit der Programmiersprache C# entwickelt und wird konstant zusammen mit den Kunden weiterentwickelt und um neue Module erweitert.

Als Mitarbeiter habe ich direkt mit den Kunden an der Umsetzung ihrer individuellen ECCOSoft Lösung und den Support dieser sowie die Installtion, Dokumentation und Aktualisierung dieser gearbeitet. Während der Arbeit haben wir zur Dokumentation und anlegen von Supportfällen sowie zur Qualitätssicherung und Versionisierung eine eigens Entwickelte ECCOSoft Lösung verwendet.

Zu meinen Tätigkeiten zählten ebenfalls die einarbeitung neuer Mitarbeiter und die Begleitung und gestaltung der Ausbildung der verschiedenen Auszubildenen in den Bereichen Systemintegration und Anwendungsentwicklung.


Einrichten und Aktualisieren von Softwarelösungen, Datenbanken etc.

Zu meinen Tätigkeiten zählte hierbei das Installieren der SQL Server Anwendungen und das Einstellen der Datenbank für die Anwendung. Dazu war es ebenfalls notwendig Client seitig die ECCOSoft Anwendung auf den Arbeitsplätzen zu installieren und die Verbindung zu dem eingerichten Server zu ermöglichen. Auf dem Server wurden außerdem Dienste für die Anwendung eingestellt. Je nach Situation bei den Kunden haben wir ebenfalls die Betriebssysteme auf den Servern eingerichtet und Aktualisierungen der Anwnedungen durchgeführt.


Eingesetzte Technologien:
  • Microsoft Windows Server 2003/2008/2012/etc.
  • Linux Server Debian/CentOS/Ubuntu
  • Microsoft SQL Server 2012/2014/2016/2019
  • Oracle SQL Server
  • Microsoft SQL Server Management Studio
  • ECCOSoft Server Dienste
  • ECCOSoft Admin Anwendung
  • ECCOSoft Initialisierungsdatei

Software-Projekt 2023:

Im 4. und 5. Semester habe ich an einem ganzjährigen Softwareprojekt teilgenommen. So konnte ich erste praxisnahe Erfahrungen sammeln. In einem Team wurden alle Schritte einer Softwareentwicklung selbstständig geplant und durchgeführt. Wobei das Wissen der vorherigen Semester kombiniert und praktisch angewendet wurde.

Ich habe an dem Projekt "Entwicklung von Informationssystemen mit SCRUM" teilgenommen. Unser Team hatte dabei die Aufgabe, eine Webseite für das Sammeln von Spenden zu erstellen. Ziel war, dass sich Spender und Spendenorganisationen einen Account anlegen können und mittels z.B. Paypal Spenden an die Organisationen entrichten können. Spender und Organisationen sollen diese Spenden in Form von Quittungen nachvollziehen können.

  • GlassFish: Anwendungsserver
  • Postgres: SQL Datenbank Anwendung

Die Webanwendung wurde mit dem Framework Java Server Faces erstellt. Hierbei handelt es sich um ein serverseitiges Framework. Java Server Faces verbirgt die Komplexität der Webanwendungsentwicklung.
